# 🧠 Compréhension Active Directory

Active Directory est un service utilisé pour gérer les utilisateurs et les ressources dans un réseau.

## Pourquoi c’est important ?

Dans une entreprise, il permet :
- de contrôler les accès
- de sécuriser les données
- d’organiser les utilisateurs

## Fonctionnement simple

- Un utilisateur appartient à un domaine
- Il est membre de groupes
- Les groupes définissent ses permissions

## Ce que j’ai compris

Active Directory facilite la gestion des accès en centralisant toutes les informations.
