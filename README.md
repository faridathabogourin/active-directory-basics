# 🏢 Active Directory Basics – Compréhension des fondamentaux

## 👋 Contexte

Dans le cadre de ma formation en informatique, j’ai étudié les bases d’Active Directory afin de comprendre comment les entreprises gèrent les utilisateurs et les accès à leurs ressources.

---

## 🎯 Objectif

- Comprendre le rôle d’Active Directory
- Identifier les éléments clés (utilisateurs, groupes, domaine)
- Comprendre le processus d’authentification

---

## 🔍 Ce que j’ai appris

- Active Directory permet de centraliser la gestion des utilisateurs
- Les utilisateurs sont organisés en groupes
- Les permissions sont attribuées via ces groupes
- L’authentification permet de contrôler l’accès aux ressources

---

## 🧠 Notions importantes

- Domaine
- Utilisateur
- Groupe
- Authentification

---

## 🧪 Exemple simple

Un utilisateur se connecte à son poste :

1. Il entre ses identifiants
2. Active Directory vérifie son identité
3. Il accède aux ressources autorisées
